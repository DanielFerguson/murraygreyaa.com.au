# Registration Application Form

## Workflow

1. A user will click 'Register Cattle' in the member dashboard.
2. They will be redirected to the registration form.
3. They will fill out the form and submit it.
4. The form will be submitted to the database.
5. The user will be redirected to the member dashboard, where they will see a success message and the new listing in a draft status.
6. An administrator user will need to approve the listing before it is published, which they will be able to do from the WordPress admin dashboard.
7. Once approved, the listing will be published and the user will be able to view it on the front-end.

## Fields
- Calf Name
- Grade
- Year Letter
- Tattoo Number
- Date of Birth
- Birth Weight
- Sex
- A.I. (boolean)
- E.T. (boolean)
- Twin (boolean)
- Calving Ease
- Colour
- Sire's Name
- Sires Registration Number Tattoo
- Dam's Name
- Dams Registration Number Tattoo
